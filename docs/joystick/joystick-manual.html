Starting to use

This code shows how to start to use the inputs helper joystick

# first you need to import joystick from inputs helper module
from inputs_helper import joystick

# then you instantiate your desired joystick, in my case the joystick 1 (index 0)
joystick = joystick(0)

# and now we will simple check if a button was pressed
if joystick["A"].pressed:
    print("Button A was pressed!")

class Joystick

The core joystick class that controls the joystick

Functions

joystick.setDeadzone(target=str, value=int)

This function will set a new deadzone to a specific target in joystick with the given value

Parameters: Target: string | Value: int (from -65535 to 65535)

Example

from inputs_helper import joystick

joystick = joystick(0)

joystick.setDeadzone("LA_UP", -7400) # new deadzone to left stick up

Return: None

joystick.vibrate(duration=int, strength=tuple(float, float) )

This function starts the joystick vibration, you can specify duration and strength

Parameters: Duration: int (0 to infinite) | Strength: tuple(float, float) (from 0.0 to 1.0)

Example

from inputs_helper import joystick

joystick = joystick(0)

joystick.vibrate(0, (1.0, 1.0)) # vibrate joystick infinitely with max strength

Return: None

joystick.stopVibration()

This function stop the joystick vibration

Parameters: None
Return: None

joystick.isVibrating()

Returns the joystick vibration status (true or false)

Parameters: None
Return: None
