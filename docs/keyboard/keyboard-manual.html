All stuff related to the keyboard you can find here.

Starting to use

This is a simple code showing how to start to use the keyboard:

# here we will import the keyboard from inputs helper module
from inputs_helper import keyboard

# you can save it in a variable if you want, but isn't necessary
# keyboard_device = keyboard

# now we will just print a message when the space key is pressed
if keyboard["Space"].activated:
    print("My Message")
